如何写规范的文档

@TOC

基本概念

  • 换行是不要钱的，只要不是连续的文本都可以在中间加一个或多个换行，无论输入多少空行，最终显示的效果也是没有空行的。
  • 本质都是文字，只是会使用一些符号来放在这一行文本的开头，来设置这一行文本的格式。
  • 注意标记符号和文本内容要有一个距离，一般是一个空格或一个换行。
  • 所有的图片都是根据链接地址寻找的，所以这张图片要先上传到网络，并得到他的链接。
